## 2024-05-22 - Auto-animating components
**Learning:** Animations that run constantly (like auto-sweepers) can be distracting or inaccessible.
**Action:** Always pause animations on focus () and hover (), ensuring keyboard users can control the component without it fighting them.
## 2024-05-22 - Auto-animating components
**Learning:** Animations that run constantly (like auto-sweepers) can be distracting or inaccessible.
**Action:** Always pause animations on focus ('onFocus') and hover ('onMouseEnter'), ensuring keyboard users can control the component without it fighting them.
